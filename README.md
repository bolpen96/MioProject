# 🎮MioProject
**Unity 기반 2D/3D 하이퍼 캐주얼 모바일 게임 프로젝트**

## 🔍프로젝트 개요
- 장르 : 하이퍼 캐주얼 + 방치형 + 던전 크롤러
- 플랫폼 : Android
- 엔진 : Unity 2021.3.37f

## 📁 폴더 구조
- `Docs/` : 기획 문서 & 로드맵
- `Assets/` : 게임 디자인 파일 (스프라이트, UI, 3D 모델)
- `Audio/` : 배경음악 & 효과음
- `Source/` : C# 코드 (Unity Scripts)
- `Builds/` : 배포 버전

## 📜 게임 기능
📌 플레이어 이동
📌 각종 작물 수확
📌 각종 아이템 사용
📌 NPC와 상호작용
📌 펫 성장
📌 AI 적 등장
📌 각종 아이템 드랍
📌 꾸미기 요소

## 🚀 실행 방법
1. Unity 2021.3.37f 설치
2. 3D(Build-in Render Pipeline) 으로 코어설정
